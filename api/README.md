# Cedar API

> An HTTP/WebSocket REST API for rapidly sending and receiving Cedar audio data

## About

This project uses [Feathers](http://feathersjs.com). An open source web framework for building modern real-time applications.

## Getting Started

Getting up and running is as easy as 1, 2, 3.

1. Make sure you have [NodeJS](https://nodejs.org/) and [yarn](https://yarnpkg.com/getting-started/install) installed.
2. Install your dependencies

    ```
    cd path/to/cedar-api
    yarn install
    ```

3. Start your app

    ```
    yarn start
    ```

## Testing

Simply run `yarn test` and all your tests in the `test/` directory will be run.
